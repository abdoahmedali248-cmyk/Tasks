# Shopping Cart Console App

A simple C# console application made to practice **Collections**.

## Collections Used
- List → to store cart items
- Dictionary → to store products and prices
- Stack → to implement Undo feature

## Features
- Add product to cart
- View cart items
- Remove item from cart
- Checkout and calculate total price
- Undo last action

## Purpose
This project is for learning and practicing **C# Collections** with simple logic.
